## Overview

This app adds a critical layer of security for users who might find complex Android prompts confusing. By adding a prominent **DANGER** warning and an **Admin Password Lock**, it ensures that no app is installed without intentional verification.

### Security Key
- **Default Password:** `?secure?`
- *Note: This ensures that even if "Install" is clicked, the action is blocked until the passkey is entered.*

## Step-by-Step Beginner Guide

### 1. Download & Install
Download the latest APK from the [Releases Page](https://github.com/editinghero/InstallerX-Security/releases). Install it on your Android device and grant all requested permissions.

### 2. Configure the Profile
To ensure the app works smoothly without constant prompts:
1. Open the app and find the **Default Profile**.
2. Click the **Pencil (Edit)** icon.
3. Scroll to **Authorizer** and select **None**.
4. Click **Save**.

- *Note Make Sure To Select the App instead of default installer and set it `Always` so this app takes over default installer*
